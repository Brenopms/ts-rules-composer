      Composes multiple rules into a single rule that runs them all and collects all errors.

      Type Parameters

      • TInput

        The type of the input to validate

      • TError = string

        The type of the error (defaults to string)

      • TContext = unknown

        The type of the context object (optional)

      Parameters

      • rules: Rule<TInput, TError, TContext>[]

        Array of rules to compose

      • options: {
            cloneContext?: boolean;
            cloneStrategy?: "json" | "shallow" | "structured";
            errorHandlingMode?: "safe" | "unsafe";
            errorTransform?: (error: unknown) => TError;
        } = {}

        Configuration options

        • OptionalcloneContext?: boolean

          Whether to clone the context for each rule (default: false)

        • OptionalcloneStrategy?: "json" | "shallow" | "structured"

          Which strategy to use when cloning the context

        • OptionalerrorHandlingMode?: "safe" | "unsafe"

          Determines how errors are handled:

          • 'safe': (default) Converts thrown errors to validation failures
          • 'unsafe': Lets errors propagate (use only in performance-critical paths)
        • OptionalerrorTransform?: (error: unknown) => TError

          Custom transformation for caught errors

      Returns Rule<TInput, TError[], TContext>

      A new rule that returns all errors from all failing rules

      Example

      const rule = every([
        validatePresence,
        validateEmail,
        validateUnique
      ]);

      const result = await rule("invalid-email");
      if (result.status === "failed") {
        console.log(result.error); // Array of all errors
      }

      Caveats

      • Rules are executed in parallel (using Promise.all)
      • All rules are executed even if some fail
      • Cloning behavior follows this priority:
        1. Uses shallowClone if options.shallowClone = true
        2. Uses structuredClone if available and options.structuredClone = true
        3. Falls back to JSON clone otherwise
